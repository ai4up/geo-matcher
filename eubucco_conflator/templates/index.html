Welcome to the Building Duplicate Labeling Tool!

The tool iterates through all new buildings that overlap an existing building and asks you to label them as duplicates or not.

The building in question is marked in red.
Existing buildings are marked in blue.
Other new buildings are also displayed for reference (but the duplicate label only applies to the building in red).

Use the buttons below to label the red building as either a duplicate, non-duplicate, or unsure.

Optional: If you wish, you can also indicate which existing building the red building duplicates by clicking on any of the existing buildings and selecting the "Duplicate" button in the popup.
